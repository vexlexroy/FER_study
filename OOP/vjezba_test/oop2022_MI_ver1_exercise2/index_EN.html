The MultiAgentSystem class is a multi-agent system that can contains 0 or more software agents. The maximum number of agents is found in the numberOfAgents constructor's argument. If the numberOfAgents is out of range then it is assumed to be 1. The generateId() static method generates unique agents' identification numbers in such a way that the generatedId static member variable increases by 1 after each creation of a new agent. If the number of agents in a multi-agent system is greater than the maximum number of agents, then the addAgent() method does nothing. The agents member variable is indexed using the agent's identification number.

The Agent abstract class represents a specific mobile or intelligent agent. The name i ID member variables define the name and identification number of the agent that cannot be changed after it is set in the ID constructor's argument (using the generateId() static method from the MultiAgentSystem class). The getAgentType() abstract method needs to be overridden in derived classes by returning a string mobile agent (if it is a mobile agent) or a string intelligent agent (if it is an intelligent agent). Calling the toString() method returns text with the agent's name and identification number printed. An example printout is below in the usage example.

The type of intelligent agent is defined by the IntelligentAgentType enumeration which can have two values REASONER and LEARNER. The value of the taskId member variable in the MobileAgent class represents ID of task (which the mobile agent performs on a remote computer) and is set in the constructor using the doRemoteTask() method which generates a random integer in the interval [1, 10].

The printSystemInfo() method in the MultiAgentSystem class prints agents that are members of a defined multi-agent system. The getAgentType() and toString() method calls are used to print the agent's type, name, and identification number. If an agent is null, then that printout is skipped.

Usage example:

MultiAgentSystem mas = new MultiAgentSystem(4);

mas.addAgent(new MobileAgent("Speedy", MultiAgentSystem.generateId()));
mas.addAgent(new MobileAgent("Agile", MultiAgentSystem.generateId()));
mas.addAgent(new IntelligentAgent("Brainy", MultiAgentSystem.generateId(), IntelligentAgentType.REASONER));
mas.addAgent(new IntelligentAgent("Smart", MultiAgentSystem.generateId(), IntelligentAgentType.LEARNER));

mas.printSystemInfo();

System.out.println("---");
System.out.println(new MobileAgent("Fast", MultiAgentSystem.generateId()));
System.out.println(new IntelligentAgent("Clever", MultiAgentSystem.generateId(), IntelligentAgentType.REASONER));
