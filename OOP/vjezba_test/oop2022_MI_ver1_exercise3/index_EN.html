Note: All classes and interfaces have visibility set to public and are in the hr.fer.oop.zad3 package.

Implement a public interface Printable that has one method print which does not receive any input parameters and returns a String object that can be printed.

Implement an abstract class PrintableList that inherits the class MyList and implements the interface Printable.

Implement the class MediaPrinter that inherits the class PrintableList and is parameterized so that it can contain all types of media. The method print returns one line for each element as shown in the usage example.

Implement the class Printer that includes two public static methods:

  • the method printVideoList which returns a String, and receives a list MyList of objects from the class Video. The method returns one line per each element of the list. The format is shown in the usage example;
  • the method printAudioList which returns a String, and receives the MyList list of objects from the class Audio. The method returns one line per each element of the list. The format is shown in the usage example;

Usage example:

Video video1 = new Video("Dune", 9300, "H265", "MP3", "3840 x 2160");
Video video2 = new Video("Don't Look Up", 8280, "H264", "AAC", "1920 x 1080");

Podcast podcast2 = new Podcast("Coding 101", 1200, "AAC", "none");
Song song1 = new Song("Free software Song", 122, "MP3", "Richard Stallman");
Song song2 = new Song("Java Life (Code Hard)", 192, "AAC", "Dev Crew");
Song song3 = new Song("I Hate the DMCA", 203, "AAC", "Joe Wecker");

MediaPrinter<Media> mediaPrinter = new MediaPrinter<>();

System.out.println(mediaPrinter.print());

MyList<Video> videos = new MyList<>();

videos.addLast(video2);
System.out.println(Printer.printVideoList(videos));
